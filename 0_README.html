BackerUp

Case

BackerUp is a purely C++ backup tool designed to be totally customizable and modular allowing modifications to fit every purpose you might need it for.

Page Structure

1) Case
2) Installation
3) Usage
4) Modification

Installation

NOTE: C++17 language standard or greater required to compile from source. Compile all source files (src/*) to build binary/executable.

Usage

Typing "help" as command displays all commands for active manager.

Managers

  • Backup Manager: handles backup creation
  • Profile Manager: handles backup profile (configuration) creation, editing

    • Modification

    Modification is easy with BackerUp. Simply add a function in the headers (include/), then define it in the corresponding C++ source file (src/). Please note that compilation will require a language standard of C++17 or greater.
